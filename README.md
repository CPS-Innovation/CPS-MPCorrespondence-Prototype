# MP Correspondence App Prototype

 A prototype of a new version of the MP Correspondence App which manages correspondence between MPs, Lords and devolved parliamentarians and the CPS using the [GOV.UK Prototype Kit](https://prototype-kit.service.gov.uk/docs/).

 > ### ⚠️ **Work in progress**
> This prototype is a work in progress and incomplete. Some parts of it may be broken and not all journeys are yet available.

 ## Get started

 1. Clone or spin up a codespace.
 2. Run ``npm run dev``
